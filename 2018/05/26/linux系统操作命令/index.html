linux系统操作命令

Posted on 2018-05-26

###linux系统操作命令

由于该系统命令较多，我就不一一阐述了，在这里我只对一些常用的做一下简单介绍。

clear : 清屏

ps: 查看shell版本

adduser:创建用户名

passwd + 用户名:设置密码

userdel -rf 用户名:删除用户

shutdown, init 0: 关机 shutdown -c:取消关机

查看帮助命令: man info –help what is

pwd: 查看当前工作路径

mkdir: 创建文件夹

touch: 创建文件

rmdir:删除空文件夹

rm:删除文件

rm -r:递归删除

rm -i:交互式删除

rm -f:强行删除

rm -rf : 删除非空的目录

ls -l :查看文件的长格式信息

ls -a:查看隐藏文件

mv: 剪切（移动，改名）

cat:查看文件内容

wget+链接地址:进入网站

cat+文件名|more(less):分屏查看

grep '<div .*>' / -R -n > result.txt 2> error.txt 输出重定向 错误重定向

ln -n 硬链接

ln -s 软连接

解压:gunzip 压缩:gzip (后缀为gz)

解压:xz -d 压缩: xz -z -9(后缀为xz)

解归档: tar -xvf (-tf,-xf)

归档:tar -cvf

chmod：改变文件执行模式

安装软件的方式: yum rpm make&make install

远程连接服务器的两种方式: sftp ssh

ps -ef |grep ：查看进程状态

kill + 进程号：停止进程

netstat -na :查看端口号

配置防火墙端口 

添加 firewall-cmd --zone=public --add-port=80/tcp --permanent （--permanent永久生效，没有此参数重启后失效） 
重新载入 firewall-cmd --reload
查看 firewall-cmd --zone=public --query-port=80/tcp
删除 firewall-cmd --zone=public --remove-port=80/tcp --permanent

这里有一些常用的命令可以做一些补充点击查看
