Django

1.windows命令下安装django服务器

检查python版本要与django版本匹配

python –version

建立虚拟环境

python -m venv 虚拟文件名

进入虚拟环境的scripts脚步激活虚拟环境

activate

退出激活

deactivate

在虚拟环境下安装django

pip install django==版本号

查看django版本

django-admin –version

在虚拟环境下建立django项目

django-admin startproject 项目名

进入django项目

cd 项目名

启动django服务器

python manage.py runserver

建立应用文件

python manage.py startapp 应用文件名

2.在linux系统下安装

mkdir hello_django

cd hello_django

建立虚拟环境venv

python3 -m venv venv

激活虚拟环境

source venv/bin/activate

更新pip

python -m pip install --upgrade pip

安装django

pip install django

创建django项目名hello_django

django-admin startproject hello_django .

3.MVT关系模型

各个模块的交互关系
Django收到HTTP请求后，依次完成下列处理：

根据URL通过URLConf模块映射到View函数，将HttpRequest对象作为参数传入。
在View函数中，获取HTTP请求的参数，通过Model访问数据库，进行业务逻辑运算得到输出数据。
然后，加载Template，根据输出数据生成页面，将HttpResponse对象返回。

生成迁移

python manage.py makemigrations 应用名

执行迁移

python manage.py migrate

创建超级管理员账号

python manage.py createsuperuser

用户跟踪的三种方式:

  • cookie(在请求头里)
  • 隐藏域(在表单里)
  • URL重写(URL里加额外信息向服务器提供身份标识，字节最多为2048)
